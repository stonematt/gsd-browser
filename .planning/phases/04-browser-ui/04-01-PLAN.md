    **Phase B — Implement tree endpoint in src/server.js:**

    Add an async `buildTree(dirPath)` function inside `createServer()` (or above it) that:
    1. Recursively scans `dirPath` using `fs.readdir` + `fs.stat`
    2. Only includes `.md` files and directories that contain `.md` files at any depth
    3. Marks directories named `.planning` or `docs` as `convention: true`
    4. Sorts: convention dirs first, then alphabetical by name
    5. Returns array of `{ name, type, path?, convention?, children? }` nodes

    Add the route:
    ```javascript
    fastify.get('/api/sources/:name/tree', {
      onSend: async (request, reply, payload) => {
        reply.header('Content-Security-Policy', MANAGEMENT_CSP);
        return payload;
      }
    }, async (request, reply) => {
      const sourceName = request.params.name;
      const sources = await listSources();
      const source = sources.find(s => s.name === sourceName && s.available);
      if (!source) return reply.code(404).send({ error: `Source not found: ${sourceName}` });
      const tree = await buildTree(source.path);
      return reply.send({ source: { name: source.name, path: source.path }, tree });
    });
    ```

    The tree response shape per research:
    ```json
    {
      "source": { "name": "my-repo", "path": "/path/to/repo" },
      "tree": [
        { "name": ".planning", "type": "dir", "convention": true, "children": [
          { "name": "ROADMAP.md", "type": "file", "path": ".planning/ROADMAP.md" }
        ]},
        { "name": "README.md", "type": "file", "path": "README.md" }
      ]
    }
    ```
