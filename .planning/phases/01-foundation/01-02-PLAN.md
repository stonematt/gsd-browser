**Create src/server.js** implementing the full server:

1. `createServer(registeredRoot)` function:
   - Creates Fastify instance with `{ logger: false }`
   - Registers `preHandler` hook setting CSP and Cache-Control headers on ALL responses:
     ```
     Content-Security-Policy: default-src 'self'; script-src 'none'; object-src 'none'
     Cache-Control: no-store
     ```
   - Add a TODO comment: `// Phase 4: change script-src 'none' to script-src 'self' when adding frontend JS`
   - Defines the CSP value as a named constant at the top of the file (not inline) so Phase 4 can update it easily

2. File serving route `GET /file`:
   - Reads `request.query.path` — if missing, return 400 JSON error
   - Calls `isPathAllowed(requestedPath, registeredRoot)` from `./filesystem.js`
   - If not allowed: return 403 with locked JSON format: `{ error: "...", status: 403, requested: requestedPath, allowed: [registeredRoot] }`
   - If allowed and path is a directory: return JSON listing `{ type: "directory", path: requestedPath, entries: [...filenames...] }` with trailing `/` on subdirectory names
   - If allowed and path is a file: read with `fs.readFile()`, set `Content-Type` via `mime-types` lookup (default `text/plain`), return content
   - For `.md` files specifically: serve as `text/plain` in Phase 1 (no rendering — per research discretionary decision)

3. `start(port, registeredRoot, options)` function:
   - Calls `fastify.listen({ port, host: '127.0.0.1' })`
   - Prints startup message: `gsd-browser serving {registeredRoot} at http://127.0.0.1:{port}\n` using `process.stdout.write()`
   - If `options.open` is true: dynamic import `open` package to open browser (wrap in try/catch — `open` may not be installed in Phase 1, that's fine)
   - Registers SIGINT handler: `process.on('SIGINT', async () => { await fastify.close(); process.exit(0); })`
   - Returns the fastify instance (useful for tests)

4. Export `{ start, createServer }`

**Update bin/gsd-browser.cjs** (replace the placeholder from Plan 01):

1. Parse args with `minimist(process.argv.slice(2), { string: ['port'], boolean: ['open', 'help', 'version'], alias: { p: 'port', h: 'help', v: 'version' } })`
2. `--version`: print `gsd-browser v{version}` from package.json, exit 0
3. `--help` or no positional args: print usage text (see research Pattern 6 for exact format), exit 0 for --help, exit 1 for no args
4. Validate positional arg is a directory: `fs.statSync(args._[0]).isDirectory()` — if not, print error and exit 1
5. Parse port: `parseInt(args.port, 10) || 4242` (default port 4242 per research recommendation)
6. Call `start(port, rootPath, { open: args.open })` with `.catch()`:
   - EADDRINUSE: `stderr.write("Port {port} in use. Try --port {port+1}\n")`, exit 1
   - Other errors: `stderr.write("Failed to start: {err.message}\n")`, exit 1

Ensure `bin/gsd-browser.cjs` stays executable (chmod +x).

```javascript
const fastify = createServer(testDir);
const response = await fastify.inject({
  method: 'GET',
  url: '/file',
  query: { path: 'valid.txt' }
});
```
