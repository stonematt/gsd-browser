<task type="auto">
  <name>Task 1: Migrate index.html to theme token references</name>
  <files>public/index.html</files>
  <action>
1. Add `<link rel="stylesheet" href="/styles/theme.css">` in `<head>` BEFORE any inline `<style>` block (should be the first stylesheet link).

2. Systematically replace ALL hardcoded hex color values in the inline `<style>` block with `var(--token-name)` references. Use the RESEARCH.md mapping table:
   - `#0d1117` -> `var(--bg-page)`
   - `#161b22` -> `var(--bg-surface)`
   - `#21262d` -> `var(--border-default)`
   - `#30363d` -> `var(--border-muted)`
   - `#c9d1d9` -> `var(--text-primary)`
   - `#f0f6fc` -> `var(--text-bright)`
   - `#8b949e` -> `var(--text-muted)`
   - `#484f58` -> `var(--text-dimmed)`
   - `#58a6ff` -> `var(--text-link)` (and `var(--status-active)` for dashboard status contexts)
   - `#3fb950` -> `var(--status-complete)`
   - `#238636` -> `var(--btn-primary-bg)`
   - `#2ea043` -> `var(--btn-primary-hover-bg)`
   - `#f85149` -> `var(--status-danger)`

3. Replace ALL `rgba(...)` values with the corresponding semi-transparent tokens:
   - `rgba(88, 166, 255, 0.15)` -> `var(--focus-ring)`
   - `rgba(88, 166, 255, 0.08)` -> `var(--hover-subtle)`
   - `rgba(88, 166, 255, 0.12)` -> `var(--active-bg)`
   - `rgba(88, 166, 255, 0.3)` -> `var(--active-ring)`
   - `rgba(88, 166, 255, 0.5)` -> `var(--active-glow)`
   - `rgba(88, 166, 255, 0.05)` -> `var(--hover-faint)`
   - `rgba(31, 111, 235, 0.12)` -> `var(--active-bg)`
   - `rgba(46, 160, 67, 0.15)` -> `var(--status-complete-bg)`
   - `rgba(46, 160, 67, 0.4)` -> `var(--status-complete-border)`
   - `rgba(248, 81, 73, 0.1)` -> `var(--status-danger-bg)`
   - `rgba(248, 81, 73, 0.3)` -> `var(--status-danger-border)`
   - Keep `rgba(0, 0, 0, 0.7)` (modal overlay) and `rgba(0,0,0,0)` (transparent) as-is — not brand colors.

4. Also check for hex values in inline JavaScript (event handlers, dynamic style assignments). Replace any hardcoded hex in JS string literals with `getComputedStyle(document.documentElement).getPropertyValue('--token-name').trim()` OR use CSS classes that reference tokens instead of inline JS style assignments.

5. Final sweep: Search for any remaining `#` followed by 3 or 6 hex chars in the file. Only non-color values should remain (e.g., hash routing `#/browse/`, HTML entities). All color hex values must be replaced.
  </action>
