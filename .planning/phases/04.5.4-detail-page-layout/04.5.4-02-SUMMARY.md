## Decisions Made

- Flat sibling DOM for phase navigator: each row is a peer div with padding-left indentation, not nested containers. This makes connector bars between rows straightforward to implement.
- Completed section collapsed by default: `splitHistoryFrontier()` determines the split; mini dot strip in header gives visual summary without expanding.
- `applyNavCollapseState()` called from within `loadProjectDetail()` so localStorage state is restored on each project navigation, not just on initial page load.
