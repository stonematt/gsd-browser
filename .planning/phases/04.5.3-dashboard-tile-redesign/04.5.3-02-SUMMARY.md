## Decisions Made

- **normalizePhaseNum()**: Strips leading zeros per segment so `"04.5.3"` maps to key `"4.5.3"` in phaseNames/phaseGoals maps. Silently failing lookup was the root cause of blank phase names in frontier list.
- **Project-relative scaling**: maxPlans is the max planCount across all history phases. Each rect width = `(planCount/maxPlans)*maxW`. This makes the largest phase always fill the max width and all others scale proportionally.
- **L-elbow SVG connectors**: Horizontal then vertical line segments connect parent rect bottom to child rect left edge in the tree topology.
- **Timeline arrow spine**: Decorative horizontal arrow runs under history strip to reinforce time direction.
- **bin/dev-server pattern**: PID file at /tmp/gsd-browser-dev.pid, start/stop/restart/status subcommands, ensures consistent dev server state across checkpoint workflows.
