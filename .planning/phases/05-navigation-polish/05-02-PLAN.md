**1. Add `resolvePath(path)` utility function** (place near other utility functions like escapeHtml):
```
function resolvePath(p) {
  const parts = p.split('/');
  const out = [];
  for (const seg of parts) {
    if (seg === '..') out.pop();
    else if (seg !== '.' && seg !== '') out.push(seg);
  }
  return out.join('/');
}
```

**3. Add `interceptMdLinks(containerEl, contextFn)` delegated click handler:**
- `containerEl`: the DOM element to listen on (e.g., `#browse-content` or `#phase-content`)
- `contextFn`: a function returning `{ currentFilePath, sourceName, viewType }` for the active document

The handler:
- Listens for `click` events via delegation: `const a = e.target.closest('a[href]')`
- If no `<a>` found, return
- Get `href = a.getAttribute('href')`
- **Skip if:** href is null, starts with `http://` or `https://` (external), starts with `#` (same-page anchor — let browser handle natively), or does not end with `.md` and does not contain `.md#`
- **For fragment-only `#` links** (heading anchors): Return early, let the browser handle native anchor scroll. The `#` check above handles this.
- `e.preventDefault()`
- Split href on `#` to separate filePart and hashFragment
- Get context from `contextFn()`
- Resolve the relative path:
  - `dir = currentFilePath.substring(0, currentFilePath.lastIndexOf('/') + 1)`
  - `resolved = resolvePath(dir + filePart)`
- **Browse view** (`viewType === 'browse'`):
  - Build hash URL: `'#/browse/' + encodeURIComponent(sourceName) + '/' + resolved`
  - `history.pushState(null, '', hashUrl)`
  - `await loadFile(sourceName, resolved)`
  - If hashFragment exists, after loadFile completes, scroll: `const el = containerEl.querySelector('#' + CSS.escape(hashFragment)); if (el) el.scrollIntoView({ behavior: 'smooth' });`
- **Detail view** (`viewType === 'detail'`):
  - The resolved path needs to be absolute (filesystem path). Since detail view files are absolute paths, `dir` is already absolute.
  - Call the existing `fetch('/render?path=' + encodeURIComponent(resolved) + '&fragment=true')` pattern and inject into `#phase-content`
  - If hashFragment, scroll to it after injection
