      - Quick-links row: only show links where `exists: true`. Clicking a quick-link navigates to `#/browse/<name>/<path>` to render that file. Use inline text link style (per Claude discretion).

    - Below cards: "Other Sources" section (if any non-GSD sources). Simple list with source name linking to `#/browse/<name>/`

    - Content rendering uses `/render?path=...&fragment=true` (unchanged)
