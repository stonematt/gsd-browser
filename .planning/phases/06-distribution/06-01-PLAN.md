**bin/gsd-browser.cjs — Auto-open config precedence:**

After loading config, before calling start(), compute `shouldOpen`:
```javascript
const explicitOpen = process.argv.includes('--open');
const explicitNoOpen = process.argv.includes('--no-open');

let shouldOpen;
if (explicitNoOpen)                    shouldOpen = false;
else if (explicitOpen)                 shouldOpen = true;
else if (config.open !== undefined)    shouldOpen = config.open;
else                                   shouldOpen = true; // default: always open
```

Pass `shouldOpen` to `start()` as `options.open`.
